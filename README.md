# Ejecutar en desarollo

1. clonar el repositorio
2. ejecutar
```
yarn install
```
3. tener Nest CLI instalado 
```
npm i -g @nestjs/cli
```
4. levantar la db 
```
docker-compose up -d
```
5. Reconstruir la BD con la semilla
```
http://localhost:3000/api/seed
```
##Stack usado
* MongoDB
* NestJs
