# Ejecutar en desarollo

1. clonar el repositorio
2. ejecutar
```
yarn install
```
3. tener Nest CLI instalado 
```
npm i -g @nestjs/cli
```
4. levantar la db 
```
docker-compose up -d
```

##Stack usado
* MongoDB
* NestJs
